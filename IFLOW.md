# LeeCommonWxMDFormatHtml 项目概述

这是一个功能丰富的在线工具集合项目，主要提供Markdown格式化、微信公众号排版、数据转换、智能文本处理、娱乐游戏和实用工具等功能。项目基于纯前端技术实现，无需后端支持，可直接在浏览器中运行。项目持续更新迭代，新增了多个智能工具和实用功能。

**核心亮点：AI驱动的智能文本格式化系统** - 新增先进的文本到Markdown智能转换工具，具备内容分析、段落识别、标题分级等AI功能。

## 核心功能模块

### 1. 微信排版工具 (wxformat/)

#### 纯净版图片排版工具
- **URL-双排图**: 使用URL链接的双列图片排版，支持在线图片快速布局
- **URL-三排图**: 使用URL链接的三列图片排版，优化多图展示效果
- **Base64-双排图**: 使用Base64编码的双列图片排版，本地图片处理
- **Base64-三排图**: 使用Base64编码的三列图片排版，本地图片处理
- **Base64-双排图(鼻子版)**: 特殊版式的双排图片排版工具

#### 文字版图片排版工具
- **图片+艺术字**: 图片与艺术字结合的创意排版
- **文字框+图片**: 文字框与图片的组合排版布局
- **朋友圈+图片**: 朋友圈风格的图片排版样式
- **图书+图片**: 图书风格与图片的排版设计

#### 图文拖拽预览工具
- **拖拽图片预览**: 支持拖拽本地图片文件进行实时预览
- **拖拽文件预览**: 拖拽各类文本文件进行预览
- **英语游戏**: 英语学习相关的互动游戏
- **智能分析**: 数据分析和智能处理工具

#### 双版本架构
- **经典版工具**: 稳定可靠的原始版本，经过充分测试
- **新版工具**: 功能增强版本，包含最新改进和优化
- **版本选择**: 用户可根据需求选择适合的版本使用

### 2. Markdown工具 (markdown/)

#### 核心Markdown工具
- **Markdown排版预览**: 实时预览Markdown排版效果，支持多种主题
- **文本转Markdown**: 将普通文本智能转换为Markdown格式（多版本工具）
- **Markdown预览**: 基础Markdown渲染预览工具
- **MD换行格式化**: 专门优化微信公众号的Markdown换行格式
- **自定义样式**: 自定义Markdown渲染HTML样式，创建独特文档外观

#### 高级工具集合
- **按钮样式工具**: 专门的按钮和样式设计工具集
  - 标题样式设计
  - 文章样式定制
  - 多种预设样式模板
- **多版本工具**: 提供多个版本的工具以满足不同需求
- **格式内容处理**: 多种format-content工具版本，适应不同文本类型
- **自定义样式优化**: 高级自定义样式工具，支持页面级和标签级定制

### 3. 智能文本格式化工具 (js/)

#### 核心AI驱动功能
- **智能文本转Markdown**: AI驱动的文本智能分析和格式化系统
- **内容结构分析**: 自动识别文本段落、标题、列表等结构元素
- **智能标题识别**: 基于上下文的精确标题分级（1-6级）
- **智能段落分割**: 通过内容跳跃度算法，准确判断段落边界
- **重点内容标记**: 多维度分析自动标记重要信息
- **列表智能识别**: 支持多种列表格式，自动识别基于上下文的列表项

#### 技术特性
- **双环境支持**: 同时支持浏览器和Node.js环境
- **中文优化**: 完整支持中文段落结构和标点符号
- **多格式支持**: 政策文件、学术文章、新闻报道、技术文档
- **格式优化**: 自动清理冗余格式，优化整体排版效果
- **版本信息**: v2.1版本，具备最新的AI算法和优化

#### 使用场景
- **文档处理**: 学术论文格式化、政府文件转换、新闻报道整理
- **内容管理**: 博客文章编写、知识库整理、会议纪要格式化
- **批量转换**: 大批量文本处理、文档库迁移、内容库格式化

### 4. 上上签工具 (shangshangqian/)

- **上上签抽签**: 互动式抽签工具，支持多种签文类型
- **签文数据库**: 丰富的签文内容，支持不同主题的抽签体验
- **界面设计**: 采用中国传统文化元素设计，界面美观

### 5. Markdown格式化工具 (mdformat/)

- **MD转公众号HTML**: 将Markdown内容转换为适合微信公众号发布的格式
- **样式优化**: 自动优化排版，提升微信公众号阅读体验
- **一键转换**: 简单快速的格式转换功能

### 6. 数据转换工具 (data-converter/)

- **CSV转JSON**: 将CSV数据转换为JSON格式，支持自定义列名和数据映射
- **JSON转CSV**: 将JSON数组数据转换为CSV格式，自动识别所有字段
- **HTML-URL提取**: 从HTML标签中批量提取URL链接和资源地址

### 7. 实用工具 (utils/)

#### 图片工具
- **图片预览器**: 拖拽图片预览工具，支持多种图片格式

#### 文本工具  
- **文本预览器**: 拖拽文件预览工具，支持各类文本格式

### 8. HTML标签工具 (htmltag/)

- **URL提取工具**: 从HTML文档中提取链接、图片地址等URL资源
- **标签解析**: HTML标签的智能解析和处理

### 9. 娱乐工具

#### 小游戏 (games/)
- **CS:GO练枪小游戏**: 基于CS:GO的瞄准训练游戏，提升反应速度和瞄准技巧

#### 英语学习游戏 (english-games/)
- **英语单词游戏**: 多种英语学习互动游戏
- **词汇练习**: 英语词汇学习和练习工具

#### 股票分析工具 (gupiao-games/)
- **股票数据分析**: 股票市场数据分析和可视化工具
- **智能分析**: 股票趋势分析和数据处理

## 开发和运行

### 本地运行
项目无需构建过程，可直接运行：

1. 克隆项目到本地
2. 使用任意HTTP服务器运行项目：
   ```bash
   # 使用Python内置服务器
   python -m http.server 8000
   
   # 使用Node.js http-server
   npx http-server
   
   # 使用PHP内置服务器
   php -S localhost:8000
   
   # 使用Live Server (VS Code扩展)
   # 右键index.html选择"Open with Live Server"
   ```
3. 访问 `http://localhost:8000` 查看主页

### 项目特点
- **纯前端实现**: 无需后端服务器，数据处理完全在客户端完成
- **响应式设计**: 完美适配各种屏幕尺寸，包括移动设备
- **模块化结构**: 各功能模块独立，便于维护和扩展
- **用户友好**: 简洁直观的界面设计，操作简单高效
- **本地隐私**: 所有工具均为客户端实现，不上传用户数据到服务器
- **即开即用**: 无需安装配置，浏览器直接访问即可使用
